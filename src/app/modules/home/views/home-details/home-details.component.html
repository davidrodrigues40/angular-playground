Welcome to my Angular project.

Hello and welcome to a fun little project which grows with each new Angular functionality I learn. I am a big fan of Angular and I am always looking for new ways to improve my skills.

What I hope you will find here is a good example of how to use Angular in a real world application. Look at the architecture and let me know what you think. Feel free to fork the repository and enjoy Angular with me.

Current Features Books

The books app is a simple example of how to use Angular call an external API and manipulate the state. It calls the Google Books API, https://www.googleapis.com/books/v1/volumes, to search for books and display the results. The state is used to store the search results as well as store the collection. The books app also uses the Angular Material library to create a nice user interface.

Bowling

The bowling app is a simple example of how to use Angular to create a simple bowling game. It uses an external API from another project in my respository. The state is used to store the game and the score. The bowling app also uses the Angular Material library to create a nice user interface.
NOTE: Offline version coming in a future project.

Chuck Norris Facts

The Chuck Norris Facts app is a simple example of how to use Angular to call an external API and manipulate the state. It calls the Chuck Norris API, https://api.chucknorris.io/jokes/random, to retrieve the jokes, I mean facts. The Chuck Norris Facts app also uses the Angular Material library to create a nice user interface.
